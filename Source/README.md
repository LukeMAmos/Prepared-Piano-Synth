# Prepared Piano Synth 

Inspired by prepared pianos , the idea of placing physical items in the strings to change the tonality of each individual  note, this synths works from that base idea by having each individual note have its own effects and synth controls. After clicking on a note the settings change to represent the effect set up of the currently selected note , from here changes to the note patch can be made and shared across notes. 

## Features 
- Individual ADSR , Reverb and distortion parameters 
- Quick Sharing of controls to all notes 
- Different synth voices available between notes

## How to install 

Use the released VST File , currently only tested on Mac devices 

## Built using 
- JUCE Framework 
- C++
